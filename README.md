## How To RUn

1. Clone this repository
   ```
   $ git clone https://github.com/feriferdinan/nusantech-test
   ```
2. Install all depedencies on the package.json
   ```
      cd nusantech-test 
      npm install
   ```
3. Run Android
   ```
   $ react-native run-android
   ```
4. Run Ios
   ```
   $ react-native run-ios
   ```
